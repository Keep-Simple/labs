== Висновок

Програма відповідає вимогам лабораторної роботи, використовуючи мову програмування C# та вбудовану базу даних SQLite. Під час виконання програма автоматично здійснює підключення до бази даних SQLite, створює таблицю "suppliers", де поле "id" має AUTOINCREMENT, поле "name" є обов'язковим (NOT NULL), а поле "address" має значення "Not provided" за замовчуванням. Після цього програма додає декілька записів у таблицю, демонструючи відпрацювання обмежень цілісності даних.
