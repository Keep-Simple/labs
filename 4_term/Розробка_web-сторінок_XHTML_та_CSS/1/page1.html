HTML and CSS

HTML (Hyper Text Markup Language) is a layout system that determines how and what elements should be placed on a web page. A style or CSS (Cascading Style Sheets, cascading style sheets) is called a set of formatting options that apply to elements of a document to change their appearance.

Types of scripts

Scripts are programs that allow you to respond to user actions. Scripts come in two forms:

  • those that run on the server, and the result of their execution comes to the browser to the user already in the finished form. These are scripts written in PHP. On it is written CMS - content management systems.
  • those that are executed directly in the user's browser. These are scripts written in JavaScript. They are most often used to make the page more comfortable and beautiful.
